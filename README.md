# SecureNotes

## About Project

SecureNotes is a local-first, privacy-focused application designed to let users create, manage, and store personal notes securely. The goal of this project is to build a lightweight, offline-capable note manager that emphasizes data protection through encryption, user authentication, and clean modular architecture.

Built using Python and MySQL, the project is structured to support scalability, encryption features (e.g. master key system), and audit logs to track sensitive operations. Ideal for developers interested in secure applications or looking to contribute to privacy-first tools.
